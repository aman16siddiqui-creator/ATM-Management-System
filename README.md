# ATM Management System (C++)

## 📌 Project Description

The **ATM Management System** is a console-based application developed in C++ that simulates basic ATM operations. The project allows users to create accounts, perform banking transactions, and manage bill payments through a simple menu-driven interface.

This project was developed to demonstrate the implementation of **functions, conditional statements, loops, and basic programming concepts in C++**.


## 🚀 Features

### Account Management
- Create a new bank account
- Store user details:
  - Name
  - CNIC
  - Age
  - City
  - Account type

### Banking Operations
- Account authentication using account number and PIN
- Withdraw money
- Deposit money
- Transfer money to another account

### Bill Payment System
Users can pay:
- Electricity bills
- Gas bills
- Internet bills


## 🛠️ Technologies Used

- **Programming Language:** C++
- **Concepts Used:**
  - Functions
  - Switch Statements
  - Loops
  - Conditional Statements
  - User Input Handling
